A bit about me

My name is Marius Szabo and I like to say that I am a front-end web designer and developer based in Cluj-Napoca, Romania.

I first started with Flash, playing around with it and making bad quality animations. Then I discovered and fell in love with Photoshop (this relationship lead to many terrible looking photos across the web and it never ended). After this, I really wanted to add some interactivity to the things I made so I ended up using Html, Css and Javascript and many other cool web technologies which I love/hate.
